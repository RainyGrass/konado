# 關於Konado

## 專案簡介

Konado是一款基於Godot Engine的視覺小說引擎框架，由開源社群驅動。Konado致力於為開發者提供高效、便捷的視覺小說開發體驗，助力創作者快速構建高質量的視覺小說作品，為創作賦能。


## 我們的目標

我們致力於自主開發一套完整的視覺小說遊戲框架，不依賴任何第三方現有解決方案。Konado的目標是打造更友好、更高效的工具，讓遊戲創作者能夠專注於故事敘述和藝術表現，而非陷入程式實現的細節中。
